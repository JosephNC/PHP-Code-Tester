Temporarily execute PHP code on any kind of page

== Description ==

This plugin allows you to test any kind of PHP code on any given page within your current WordPress instance. 
You can execute code without the need of editing or manipulating files. 
The purpose of this plugin is to make debugging and testing easier than ever. 
Down below, you will find further details, as well as some use-cases.

= Caution! =

Please only use this plugin if you are familiar with PHP and WordPress. It is very powerful and allows every administrator to execute PHP directly code on your website.

= Features =

* Create, Delete Update, Search and Retrieve users via external webhooks on your website
* Add, update and delete user meta on all of the user action webhooks

= For devs =

This plugin is specifically designed for developers who are familiar with PHP and WordPress.

== Installation ==

1. Activate the plugin
2. Once activated, you will see a fixed container at the bottom of the page.
3. There, you can add your code and execute it
